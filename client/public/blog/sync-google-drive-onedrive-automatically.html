How to Sync Google Drive with OneDrive Automatically

Last updated: July 2026 · 4 min read

Keeping files in sync between Google Drive and Microsoft OneDrive is a common need — especially for people who work in environments that use both Google Workspace and Microsoft 365. Doing it manually every time is slow and easy to forget.

This guide shows how to set up an automatic sync between Google Drive and OneDrive using TERA's scheduled tasks feature.

What "sync" means here

TERA's sync is a one-way scheduled transfer: new files added to a source folder are automatically copied to a destination folder on the schedule you define. If you need files to flow from Google Drive to OneDrive, you set up a task with Drive as the source and OneDrive as the destination.

It's not a real-time bidirectional sync (files deleted on one side don't delete on the other), but for most use cases — like keeping a project folder backed up, or making work files available in both ecosystems — it covers the need.

Step-by-step setup

  1. Create an account at mytera.app if you don't have one
  2. Go to Integrations and connect your Google Drive account
  3. Connect your Microsoft OneDrive account in the same section
  4. Go to Tasks in the sidebar
  5. Click New task
  6. Set the source to a Google Drive folder and the destination to an OneDrive folder
  7. Choose your schedule: daily, weekly, or a custom interval
  8. Save — TERA will run the sync automatically from that point on
Tip: You can create multiple tasks pointing to different folder pairs. For example, one task that syncs your work projects folder and another for documents.

What gets transferred

On each run, TERA checks for files in the source folder that don't yet exist in the destination and transfers them. Files that already exist are skipped to avoid unnecessary re-transfers.

Google Workspace files (Docs, Sheets, Slides, Forms) are automatically converted to their Microsoft equivalents (docx, xlsx, pptx) when transferred to OneDrive.

Transfer history

Every sync run is logged in the Operations section, so you can see exactly what was transferred, when, and whether it succeeded. If a file fails to transfer, it shows up there with the reason.

Can I sync in the other direction?

Yes — OneDrive to Google Drive works the same way. Set the source to an OneDrive folder and the destination to a Google Drive folder, and TERA transfers files in that direction instead.

Other sync combinations

The same task system works for any combination of supported providers: Google Drive, Dropbox, OneDrive, Box and Amazon S3. So you can also set up syncs like:

  • Dropbox → Google Drive
  • Google Drive → Amazon S3 (for backups)
  • OneDrive → Dropbox
  • Box → Google Drive
